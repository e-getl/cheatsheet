HTML & CSS Cheatsheet

Learn HTML and CSS with this cheatsheet. This cheatsheet is designed to help you quickly reference HTML and CSS syntax, properties, and values.

Basic HTML

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It describes the structure of a webpage using elements and tags.

HTML tags

Tag Description
<!DOCTYPE html> Defines the document type and version of HTML.
<html> The root element of an HTML page.
<head> Contains meta-information about the document.
<body> Contains the content of the document.
<link rel="stylesheet" href="styles.css"> Links an external CSS stylesheet to the HTML document.
<div> Defines a division or section in an HTML document.
<span> An inline container used to mark up a part of a text, or a section in a document.

CSS Attributes

Attribute Description
background-color Sets the background color of an element.
color Sets the text color of an element.

IDs & Classes

IDs and classes are used to select and style HTML elements with CSS.

HTML Tags

Tag Description
<id=""> Specifies a unique id for an HTML element, can only be used for one HTML element within the page.
<class=""> Specifies a class for an HTML element, can be used for multiple HTML elements within the page.

CSS attributes

Attribute Description
# #id in CSS, can only be used for one element within the page
. .class in CSS, can be used for multiple elements within the page

Typography

Typography refers to the style and appearance of printed matter. In web design, it involves the use of fonts, sizes, spacing, and other text properties.

HTML Tags

Tag Description
<h1> to <h6> Defines HTML headings, from largest<h1> to smallest<h6>, do not skip headings .
<p> Defines a paragraph.
<strong> Defines important text and makes it bold.
<em> Defines emphasized text.

CSS Attributes

Attribute Description
font-family Specifies the font for an element.
font-size Specifies the size of the font.
font-weight Specifies the weight of the font.
text-transform Specifies how to capitalize an element's text like capitalize,uppercase,lowercaselowercase,none
text-align Sets the horizontal alignment of the inline-level content inside a block element or table-cell box likecenter,start,end,justify

Tables

Tables are used to display data in a structured format. They consist of rows and columns.

HTML Tags

Tag Description
<table> Defines a table.
<thead> Defines the header section of a table.
<tr> Defines a row in a table.
<th> Defines a header cell in a table.
<tbody> Defines the body section of a table.
<td> Defines a standard cell in a table.
<tfoot> Defines the footer section of a table.
Tag Description
colspan Specifies the number of columns a cell should span.
rowspan Specifies the number of rows a cell should span.

CSS Attributes

Attribute Description
border Sets the border of a table - including width, style and color
padding Defines the space between the cell content and its border, apply to <td>and<th>
border-spacing Defines the space between cells in a table, use on<table>property

Lists

Lists are used to group related items together. There are two types of lists: ordered and unordered.

HTML Tags

Tag Description
<ul> Defines an unordered list.
<ol> Defines an ordered list.
<li> Defines a list item.

Images

Images are used to display pictures on a webpage. They can be linked to other pages or websites.

HTML Tags

Tag Description
<img src="" alt=""> Defines an image, with the source and alternative text.

Links

Links are used to navigate between pages or websites. They can be styled with CSS.

HTML Tags

Tag Description
